# Smart Scientific Calculator Logic (Delphi)

## Overview
A high-precision mathematical engine developed in Delphi. This project demonstrates clean coding practices, exception handling (like division by zero), and the implementation of core arithmetic and scientific functions.

## Features
- **Core Operations:** Addition, Subtraction, Multiplication, and Division.
- **Advanced Logic:** Square root and power calculations.
- **Error Handling:** Robust validation to prevent system crashes during invalid operations.
- **Object-Oriented Design:** Logic is encapsulated within a dedicated `TCalculator` class for modularity.

## Technical Skills
- **Delphi Pascal:** Class structures and function implementation.
- **Exception Management:** `try...except` blocks for safe execution.
- **Mathematics in Programming:** Precision handling with `Double` data types.

## How to use
This is a logic unit (`.pas`). It can be easily integrated into any Delphi VCL or FMX UI application.
